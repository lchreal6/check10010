# 查询中国联通流量话费的JSBox脚本

## 使用方法

1. 修改脚本的`USER_ID`的值为你手机号码
2. 若未使用过支付宝中国联通小程序的需先进去授权登录一下，本脚接口使用的是支付宝小程序抓包得到的接口
